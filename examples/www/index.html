Bem-vindo ao WebServ, um servidor HTTP completo em C++98

Este servidor foi desenvolvido 100% em C++ como parte do currículo da escola 42, focando em performance, modularidade e robustez. Este é um servidor HTTP completo em C++98, não-bloqueante, capaz de servir websites estáticos, processar CGI e gerenciar múltiplas conexões simultâneas.

O que é um site web?

Estrutura

Um site é composto por páginas interligadas que compartilham um propósito comum e navegação fluida.

Conteúdo

Textos, imagens, vídeos e interações formam a base de um site moderno e atrativo.

Domínio

O endereço único (como webserv.com) permite acesso global e identificação na internet.

Design

Uma boa interface garante usabilidade, acessibilidade e experiência positiva ao usuário.

Tecnologia

Sites podem ser estáticos ou dinâmicos, com backend em linguagens como C++, Python ou Node.js.

Interatividade

Formulários, uploads, animações e feedback visual tornam o site mais envolvente.
